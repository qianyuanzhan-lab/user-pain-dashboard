## 四、数据质量说明

### 4.1 数据来源合规性

| 数据源 | 采集方式 | 合规状态 |
|--------|----------|----------|
| App Store | RSS API（官方接口） | ✅ 完全合规 |
| Google Play | 公开 API | ✅ 完全合规 |
| Hacker News | Algolia 搜索 API | ✅ 完全合规 |

### 4.2 数据处理流程

```
原始评论 → 去重/清洗 → 痛点提取 → 聚类合并 → AI评分 → 排序输出
```

### 4.3 样本审核标准

- ❌ 排除：纯情绪宣泄、垃圾广告、无意义重复
- ❌ 排除：长度 < 20 字的评论
- ✅ 保留：有具体功能诉求、有使用场景描述、有对比说明
