## 该簇的全部代表诉求（representative_statements）

1. 系统应该提供内置的日历应用，避免用户安装第三方工具
2. 日历类应用应该避免功能膨胀，保持简洁
3. 希望有一个只显示年历的 Android 应用，不包含事件、提醒等多余功能
4. 希望日历视图支持显示农历日期
5. 日历应用应该提供设置，控制新建日程是否默认开启提醒
